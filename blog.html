Document Object Model (DOM) is like a tree or interface for web documents which allow us to manipulate the page with programming language.

Purpose of DOM:
Any element or node of the document tree can be accessed by programming language. We can easily change any style, attribute, value etc dynamically without editing in the css or html file directly. It allows us to make a website powrfully interactive by defining a standard fo accessing web pages.

I can select HTML element by their tag/class/id name. And there are 2 other way to access with querySelector method. There are 5 different ways to select HTML elements using DOM. They are:

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

Example: document.getElementById("id name")

Event bubble is a method of event propagation in the document tree. This happens when any element receives an event and bubble up to its parent element or parent node. Through bubbling any parent or children can be accessed.

A function which is passed as an argument to another function is called a callback function.

When I need a function to run only after a specific event occurs then I will use callback function. Events like mouse click, mouse over etc.
